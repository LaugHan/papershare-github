When Is Collective Intelligence a Lottery?
Multi-Agent Scaling Laws for Memetic Drift

arXiv:2603.24676 cs.AI 2026-03-25

📝 摘要

由大语言模型（LLM）驱动的多智能体系统正越来越多地部署在影响决策的环境中。然而，这些系统的表现究竟反映的是集体推理、系统性偏见还是单纯的随机性，目前仍不清楚。

本文引入最小模型——量化单纯形闲聊（QSG），揭示了共识形成的机制：将协议形成的微观起源追溯到相互的上下文学习（mutual in-context learning）。

在 QSG 中，智能体维持内部信念状态，但从彼此的采样输出中进行学习。因此，一个智能体的任意选择成为下一个智能体的证据，并可能复合向一致方向发展。论文将此称为模因漂移（Memetic Drift）。

QSG 预测了两种状态的交叉：
• 漂移主导区：共识实际上相当于彩票
• 选择主导区：弱偏见被放大并塑造结果

推导了漂移诱导极化的缩放定律，并在模拟和 LLM 实验中验证。

🎯 方法动机

核心问题

Multi-Agent LLM 系统的集体决策，究竟是：
• 真正的集体推理？
• 系统性偏见的放大？
• 纯粹的随机性？

已有观察

命名游戏实验表明，即使没有先验偏好，LLM 群体也能迅速打破对称性达成共识。

研究空白

  • 缺乏对共识形成机制的理解
  • 没有量化分析框架
  • 无法预测系统何时是"彩票"何时是"选择"

核心假设

相互的上下文学习是共识形成的关键——一个智能体的任意选择会成为下一个智能体的"证据"，形成正反馈放大。

⚙️ QSG 模型设计

核心概念：Quantized Simplex Gossip

  • Q: 量化位数（信念空间的离散化精度）
  • S: 单纯形（信念状态空间）
  • G: 闲聊机制（智能体间的采样输出交换）

工作原理

智能体 i 维护信念状态 θᵢ
智能体 j 采样输出 xⱼ ~ p(·|θⱼ)
智能体 i 更新 θᵢ ← f(θᵢ, xⱼ) ← 通过上下文学习

系统性结果

关键参数

参数含义对漂移的影响
N群体规模N ↑ → Drift ↓
Q量化位数Q ↑ → Drift ↑
η适应率η ↑ → Selection ↑
T采样温度T ↑ → Drift ↑

📈 Scaling Laws

漂移诱导极化公式

Polarization ∝ f(N, Q, η, T)

关键交叉点：Qη/N = constant

推导的 Scaling Laws

  • 漂移随群体规模 N 的衰减
  • 漂移随量化位数 Q 的增长
  • 漂移随适应率 η 的非线性关系
  • 漂移随内部不确定性（采样温度 T）的增长

实验验证

两种验证方式：
1. QSG 模拟 → 理论预测吻合
2. LLM 命名游戏实验 → 真实系统验证

🔬 对 LLM 诊断的价值

与 Self-Consistency 的关系

方法本质风险
Self-Consistency多路径采样+投票可能陷入 Drift 主导区
本文理论量化分析 Drift vs Selection提供判断框架

诊断应用

  • 置信度校准：在高 Drift 场景下，投票结果置信度低
  • 采样策略设计：根据 Qη/N 判断是否需要更多 diversity
  • Multi-Agent 一致性检测：区分真实共识 vs 随机漂移

设计原则

降低 Drift 的方法：
• 增加群体规模 N
• 减少每轮通信量（降低 Q 的影响）
• 引入显式偏见信号（进入 Selection 主导区）

🎯 核心总结

一句话概括：揭示 Multi-Agent LLM 系统中的模因漂移机制，提供从"随机彩票"到"系统性选择"的量化分析框架

Pipeline（3步）

  1. 智能体维持信念状态 → 通过采样输出进行相互学习
  2. 分析 Qη/N 比率 → 判断 Drift vs Selection 主导
  3. 推导极化 scaling laws → 预测共识质量

对诊断方向的贡献

提供了区分真实一致性 vs 随机一致性的理论工具：

  • 当系统处于 Drift 主导区时，投票结果不可靠
  • Self-Consistency 等方法的成功取决于进入 Selection 主导区
  • 为置信度校准提供物理直觉
