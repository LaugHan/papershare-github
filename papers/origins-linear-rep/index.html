线性表示的起源：从理论到实验

arXiv 2403.03867 | next-token prediction + gradient descent

📖 摘要

近期研究表明，高层语义概念在大语言模型的表示空间中以"线性"方式编码。

本文研究这种线性表示的起源：引入简单的潜在变量模型来形式化 next-token prediction 的概念动态。

核心证明：① 线性表示从 log-odds 匹配中自然产生；② next-token prediction 目标结合梯度下降的隐式偏差导致线性表示涌现。

在 LLaMA-2 上的实验验证了理论发现。

🎯 核心问题与假设

科学问题：线性表示是如何产生的？是训练中自然涌现还是巧合？
核心假设：线性表示是 next-token prediction 目标 和 梯度下降隐式偏差 共同作用的自然结果。
建模假设：
  • 上下文句子反映潜在二元概念变量
  • 下一个 token 也反映潜在二元概念变量
  • 通过梯度下降学习，相似的概念向量会趋向对齐

🔧 方法：潜在变量模型

模型结构

Context (x) → 潜在概念 z ∈ {0,1}^K → Next Token (y)
z: K个二元概念变量 | f: 双射映射函数

梯度下降演化方程

u_{t+1} = u_t + η·L(t)·v_t
v_{t+1} = v_t + η·L(t)·u_t

其中 u=嵌入向量, v=unembed向量, η=学习率, L(t)=损失函数

关键结论

  • 内积 ⟨u_t, v_t⟩ 的变化量始终为正
  • 对齐程度 cos(u, v) → 1（单调递增）
  • 不同概念向量保持近似正交

📊 关键发现

1. Log-odds 匹配 → 线性结构
当学习的概率 log-odds 匹配时，线性结构在子空间中自然涌现。
2. 梯度下降 → 向量对齐
梯度下降的隐式偏差导致同概念向量趋于完全对齐（cos→1），异概念保持正交。
3. LLaMA-2 实验验证
同一概念的 steering vectors 有非平凡的对齐度，不同概念的 steering vectors 几乎正交。

🔗 理论贡献

统一解释链：

Next-token Prediction Objective
↓
Gradient Descent Implicit Bias
↓
Linear Representations Emerge
↓
↑ ↑ ↑
同概念对齐 | 异概念正交

这是首次从理论上证明：线性表示不是巧合，而是训练的必然结果。

⚠️ 局限性

  • 模型简化：潜在变量模型可能过于简化真实LLM
  • 二元假设：真实概念可能是多元或连续的
  • 实验范围：主要在LLaMA-2验证，需更多模型验证
